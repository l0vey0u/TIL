- 귀무가설에서 나오는 얘기인데
음 찾아보니 여기 좋은것 같다.

- 전자를 1종오류, 후자를 2종 오류라 하는데 1종오류는 실제론 false 인데 탐지 혹은 실험이 positive 인 상황이다.
2종 오류는 반대고 실제로 positive인데 탐지 혹은 실험이 false 인 상황이다.
