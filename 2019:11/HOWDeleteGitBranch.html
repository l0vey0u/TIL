# 깃허브 브랜치 간단히 삭제하기

내 RESUME 레포가 Github Pages 기능이 필요없어졌는데 Github Pages 기능을 웹 ui 내에서 끄려니 비 활성화 되있어서 당황했다.

그냥 간단히 gh-pages 브랜치를 제거하면 해결되는 문제였고 제거는 더욱 간단했다.

# 제거 명령어

--delete

  • git push origin --delete {branch name} {branch name} 에 해당하는 브랜치를 적어주면 끝이다.
